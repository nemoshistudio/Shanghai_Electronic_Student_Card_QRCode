与普通二维码生成器主要差异在于：

  1. 为 二维码 内 字节字符加入 UTF-8 的 BOM 头，以适配 数字哨兵 等需要 BOM 头才能正常读取的设备
  
  2.将生成的二维码规格限制于 version 4 ，纠错等级 设定为 15% ，使各项规格保持与 学生证实体卡 二维码相同
